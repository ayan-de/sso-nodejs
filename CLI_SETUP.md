# CLI Setup Tool - `@thisisayande/create-sso`

An interactive CLI tool that sets up SSO in your project by asking questions and generating configuration.

---

## Installation & Usage

```bash
# Run at your project root
npx @thisisayande/create-sso
```

## CLI Options

```bash
# Interactive mode (default)
npx @thisisayande/create-sso

# Non-interactive mode with preset config
npx @thisisayande/create-sso --config ./sso.config.json

# Skip code generation, only update .env
npx @thisisayande/create-sso --env-only

# Use preset template
npx @thisisayande/create-sso --template nextjs-fullstack

# Dry run - show what would be generated
npx @thisisayande/create-sso --dry-run

# Force overwrite existing files
npx @thisisayande/create-sso --force

# Verbose output
npx @thisisayande/create-sso --verbose
```

## Generated Code Examples

### Next.js SSO Client Wrapper (`lib/sso.ts`)

```typescript
import { SSOClient } from '@thisisayande/sso-client';

export const ssoClient = new SSOClient({
  appId: process.env.SSO_APP_ID!,
  appSecret: process.env.SSO_APP_SECRET!,
  ssoUrl: process.env.SSO_BASE_URL!,
});

export const getLoginUrl = (
  provider: string = 'google',
  redirectTo?: string
) => {
  return ssoClient.getLoginUrl({
    redirectUri: redirectTo || `${window.location.origin}/auth/callback`,
    state: crypto.randomUUID(),
    provider,
  });
};

export const validateToken = async (token: string) => {
  return await ssoClient.validateToken(token);
};
```

### React Login Component (`components/LoginButton.tsx`)

```tsx
'use client';

import { getLoginUrl } from '@/lib/sso';

export function LoginButton() {
  const providers = [
    { id: 'google', label: 'Google', icon: '/icons/google.svg' },
    { id: 'github', label: 'GitHub', icon: '/icons/github.svg' },
  ];

  return (
    <div className="login-buttons">
      {providers.map((provider) => (
        <button
          key={provider.id}
          onClick={() => (window.location.href = getLoginUrl(provider.id))}
          className="provider-button"
        >
          <img src={provider.icon} alt={provider.label} />
          Continue with {provider.label}
        </button>
      ))}
    </div>
  );
}
```

### Express Middleware Setup (`sso.config.ts`)

```typescript
import { createSSOMiddleware, RedisSessionStore } from '@thisisayande/sso';
import Redis from 'ioredis';

const redis = new Redis({
  host: process.env.REDIS_HOST,
  port: parseInt(process.env.REDIS_PORT || '6379'),
  db: parseInt(process.env.REDIS_DB || '0'),
});

export const ssoMiddleware = createSSOMiddleware({
  sessionStore: new RedisSessionStore(redis),
  baseUrl: process.env.SSO_BASE_URL!,
  providers: {
    google: {
      clientId: process.env.GOOGLE_CLIENT_ID!,
      clientSecret: process.env.GOOGLE_CLIENT_SECRET!,
    },
    github: {
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
    },
  },
});
```

**Note:** `@thisisayande/sso` re-exports all components from internal packages. You only need to install `@thisisayande/sso`!

## CLI Features

### ✅ What It Does

1. **Interactive wizard** - Step-by-step questions with validation
2. **Smart defaults** - Suggests values based on project structure
3. **Env file detection** - Finds existing .env files automatically
4. **Safe appending** - Adds SSO vars without overwriting existing ones
5. **Secret generation** - Generates secure app secrets and JWT keys
6. **Code scaffolding** - Generates boilerplate code for your framework
7. **Multiple frameworks** - Supports React, Next.js, Express, Vue, etc.
8. **Config file support** - Non-interactive mode with JSON config

### 🔒 Security Features

- Generates cryptographically secure secrets
- Warns about weak/default values
- Validates OAuth credentials format
- Suggests secure cookie settings for HTTPS

### 📦 File Structure

```
your-project/
├── .env                    # Updated/created with SSO vars
├── .env.example            # Created as reference
├── keys/
│   ├── private.pem         # Generated JWT private key
│   └── public.pem          # Generated JWT public key
├── sso.config.json        # Optional config file
├── lib/
│   └── sso.ts            # Generated client wrapper (optional)
├── components/
│   └── LoginButton.tsx    # Generated login component (optional)
└── app/
    └── auth/
        ├── callback/
        │   └── page.tsx   # Generated callback handler (optional)
        └── login/
            └── page.tsx   # Generated login page (optional)
```

## Updated Environment Loading

All SSO packages read from environment variables:

```typescript
// packages/core/src/config/loader.ts
export function loadSSOConfig(): SSOConfig {
  return {
    server: {
      port: parseInt(process.env.SSO_PORT || '5000'),
      baseUrl: process.env.SSO_BASE_URL!,
    },
    jwt: {
      privateKey: fs.readFileSync(process.env.JWT_PRIVATE_KEY_PATH!, 'utf8'),
      publicKey: fs.readFileSync(process.env.JWT_PUBLIC_KEY_PATH!, 'utf8'),
      issuer: process.env.JWT_ISSUER || 'sso',
      audience: (process.env.JWT_AUDUENCE || 'sso').split(','),
      expiresIn: process.env.JWT_EXPIRES_IN || '24h',
    },
    storage: {
      type: (process.env.STORAGE_TYPE || 'memory') as StorageType,
      redis: {
        host: process.env.REDIS_HOST || 'localhost',
        port: parseInt(process.env.REDIS_PORT || '6379'),
        password: process.env.REDIS_PASSWORD,
        db: parseInt(process.env.REDIS_DB || '0'),
      },
      // ... postgres, mongodb configs
    },
    providers: {
      google: {
        clientId: process.env.GOOGLE_CLIENT_ID!,
        clientSecret: process.env.GOOGLE_CLIENT_SECRET!,
      },
      github: {
        clientId: process.env.GITHUB_CLIENT_ID!,
        clientSecret: process.env.GITHUB_CLIENT_SECRET!,
      },
      // ... other providers
    },
    applications: [
      {
        id: process.env.APP1_ID!,
        name: process.env.APP1_NAME!,
        secret: process.env.APP1_SECRET!,
        redirectUrl: process.env.APP1_REDIRECT_URL!,
      },
    ],
    security: {
      csrf: { enabled: process.env.CSRF_ENABLED === 'true' },
      rateLimit: {
        windowMs: parseInt(process.env.RATE_LIMIT_WINDOW_MS || '900000'),
        max: parseInt(process.env.RATE_LIMIT_MAX || '100'),
      },
      cookies: {
        secure: process.env.COOKIE_SECURE === 'true',
        sameSite: (process.env.COOKIE_SAME_SITE || 'lax') as 'strict' | 'lax' | 'none',
      },
    },
  };
}
```

## Quick Start Command

```bash
# Start a new project with SSO
npx create-next-app@latest my-app && cd my-app
npx @thisisayande/create-sso --template nextjs-fullstack

# Add SSO to existing project
cd existing-project
npx @thisisayande/create-sso

# Set up with config file
npx @thisisayande/create-sso --config ./sso.config.json
```
